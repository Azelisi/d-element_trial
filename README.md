# Тестовое задание - de-landingpage

* Установка - `npm i`
* Запуск локального сервера - `gulp`
* Сборка проекта без запуска локального сервера - `gulp build`
## Разработка ведётся в директории `app`
## Итоговый код попадает в директорию `dist`
